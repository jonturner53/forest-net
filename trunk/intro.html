CSE 542 Source Code

This page provides links to source code for students in CSE 542. There are two major categories of programs: data structures and algorithms. The various groups of programs within each category are described briefly below.

Data Structures

  • Basic. The programs in this group implement lists, collections of lists and partitions.
  • Graphs. The programs in this group implement implement several kinds of graphs, including undirected graphs, directed graphs and flow graphs (used to solve network flow problems). An edge-weighted version of each type is also provided.
  • Heaps. The programs in this group implement several types of heaps, including d-heaps, leftist heaps and Fibonacci heaps.
  • Search Trees. The programs in this group implement several types of search trees, including simple (unbalanced) binary search trees and self-adjusting binary search trees.
  • Advanced. The programs in this group implement various advanced data structures including path sets and dynamic trees.

Algorithms

  • Miscellaneous. The programs in this group implement miscellaneous functions, including breadth-first search, nearest-common ancestor and random graph generation.
  • Minimum Spanning Tree. The programs in this group implement algorithms for the minimum spanning tree problem. They include Kruskal's algorithm, Prim's algorithm and the Round Robin algorithm.
  • Shortest Paths. The programs in this group implement algorithms for the shortest path problem. They include Dijkstra's algorithm, the breadth-first scanning algorithm and Floyd's algorithm.
  • Maximum Flow. The programs in this group implement algorithms for the maximum flow problem. They include the shortest augmenting path algorithm, Dinic's algorithm and Sleator & Tarjan's algorithm.
  • Minimum Cost Flow. The programs in this group implement algorithms for the minimum cost flow problem.
  • Matchings. The programs in this group implement algorithms for the matching problem.

Miscellany

  • Miscellaneous Utilities. Like it says. .h file, .c file.
The programs are organized into several directories and sub-directories. At the top level, there are directories called include, dataStructures, algorithms, misc and lib. There is a makefile at the top level, which recursively invokes makefiles in the sub-directores and builds several libraries: lib-ds, lib-ga, lib-misc, which are stored in the lib sub-directory. The top-level makefile includes some definitions that should be modified to reflect the full path name of the directory containing the top-level makefile. A "make clean" command will reset all directories and a "make all" command will re-build all the programs.

The web pages provide a convenient way to browse the code and get an understanding of the programs. A tarfile of the entire directory structure is also available here. Download the tarfile (by right-clicking on the above link) to a unix or linux system and move it to the desired file system location, then type "tar xf cse542tarfile" to get your own working copy of all the programs.
