Shortest Path Algorithms

This page provides links to source code for various algorithms for the shortest path problem. There is an associated makefile which can be used to compile all the programs. The makefile assumes a particular directory structure. Specifically, the various groups of algorithms are stored in sub-directories with a common parent. The parent of this directory is assumed to have a child directory called data_structures, where various data_structures can be found.

The links at left provide access to all the same files that can be accessed through the links below. Once one becomes familiar with the various files, these shortcut links provide a more convenient way to access the code.

  • SPT Command. Reads an input graph and invokes an spt algorithm, specified on the command line. Source.
  • SPT Repeat Command. Generates random graphs and computes their spt using a specified algorithm. Source.
  • All Pairs Command. Reads an input graph and computes solution to all pairs shortest path problem, using algorithm, specified on the command line. Source.
  • All Pairs Repeat Command. Generates random graphs and computes the solution to the all pairs problem using a specified algorithm. Source.
