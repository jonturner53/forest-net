Min Cost Flows

This page provides links to source code that implements algorithms for the min cost flow problem. There is an associated makefile which can be used to compile all the programs. The makefile assumes a particular directory structure. Specifically, the various groups of algorithms are stored in sub-directories with a common parent. The parent of this directory is assumed to have a child directory called data_structures, where various data_structures can be found.

The links at left provide access to all the same files that can be accessed through the links below. Once one becomes familiar with the various files, these shortcut links provide a more convenient way to access the code.

  • Cycle Reduction Algorithm. Source.
  • Least Cost Augmenting Path Algorithm. Source.
  • Min Cost Flow Command. Reads an input graph and computes a min cost flow using a specified algorithm Source.
  • Min Cost Flow Repeat Command. Generates random graphs and computes a min cost flow using a specified algorithm. Source.
  • Min Cost Flow Checker. Reads a flow graph from stdin and checks to see if the defined flow is a minimum cost flow. It prints messages whenever it finds something incorrect. If the flow is correct, there is no output. Usage: check. Source.
