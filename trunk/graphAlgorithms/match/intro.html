Matching Algorithms

This page provides links to source code that implements algorithms for the matching problem. There is an associated makefile which can be used to compile all the programs. The makefile assumes a particular directory structure. Specifically, the various groups of algorithms are stored in sub-directories with a common parent. The parent of this directory is assumed to have a child directory called data_structures, where various data_structures can be found.

The links at left provide access to all the same files that can be accessed through the links below. Once one becomes familiar with the various files, these shortcut links provide a more convenient way to access the code.

  • Alternating Path Algorithm for Unweighted Matching in Bipartite Graphs. Source.
  • Faster Version of Alternating Path Algorithm for Unweighted Matching in Bipartite Graphs. This version is organized to avoid the per search initialization overhead that can slow down the basic version. Source.
  • Bipartite Matching Algorithms using Flows. Implements two variants, one for max size matchings, one for max weight matcings. Source.
  • Matching Command. Reads an input graph and computes a maximum matching using a specified algorithm Source.
  • Matching Repeat Command. Generates random graphs and computes a maximum matcing using a specified algorithm. Source.
