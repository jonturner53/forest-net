Basic Data Structures

This page provides links to source code for some basic data structures. The data structures are implemented as C++ classes, so there is a header file (.h) and source file (.c) for each data structure. Most data structures also have an associated driver, which is a command that can be used to exercise the various data structure operations and verify that the code works as expected.

A makefile is provided. It assumes a certain directory structure. In particular, the different data structure directories all have a common parent and all contain a directory called dvr that contains driver programs.

The links at left provide access to all the same files that can be accessed through the links below. Once one becomes familiar with the various files, these shortcut links provide a more convenient way to access the code.

  • Simple List. Linked list defined on integers [1,n]. Class definition (.h file). Code for operations (.c file). Driver.
  • Doubly Linked List. Linked list defined on integers [1,n]. Class definition (.h file). Code for operations (.c file). Driver.
  • Collection of lists. Collection of doubly-linked circular lists that form a partition on integers [1,n]. Class definition (.h file). Code for operations (.c file). Driver.
  • Partition Data Structure. Data structure representing a partition on a set of items numbered [1,n]. Also know as disjoint set data structure or union-find data structure. Class definition (.h file). Code for operations (.c file). Driver.
  • d-Left Hash Table. Data structure implementing a hash table that stores (key,value) pairs where the values are positive integers in a bounded (but large) range. Class definition (.h file). Code for operations (.c file). Driver.
