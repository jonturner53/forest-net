Advanced data structures

This page provides links to source code for some advanced data structures. The data structures are implemented as C++ classes, so there is a header file (.h) and source file (.c) for each data structure. Most data structures also have an associated driver, which is a command that can be used to exercise the various data structure operations and verify that the code works as expected.

A makefile is provided. It assumes a certain directory structure. In particular, the different data structure directories all have a common parent and all contain a directory called dvr that contains driver programs.

The links at left provide access to all the same files that can be accessed through the links below. Once one becomes familiar with the various files, these shortcut links provide a more convenient way to access the code.

  • Path Set. Data structure representing a set of paths. The vertices in the paths define a partition on a set of items numbered [1,n]. Each item has an associated cost. Class definition (.h file). Code for operations (.c file). Driver.
  • Dynamic Trees. Data structure representing a set of trees. The vertices in the trees define a partition on a set of items numbered [1,n]. Each item has an associated cost. Class definition (.h file). Code for operations (.c file). Driver.
