Heaps

This page provides links to source code for various types of heaps. The data structures are implemented as C++ classes, so there is a header file (.h) and source file (.c) for each data structure. Most data structures also have an associated driver, which is a command that can be used to exercise the various data structure operations and verify that the code works as expected.

A makefile is provided. It assumes a certain directory structure. In particular, the different data structure directories all have a common parent and all contain a directory called dvr that contains driver programs.

The links at left provide access to all the same files that can be accessed through the links below. Once one becomes familiar with the various files, these shortcut links provide a more convenient way to access the code.

  • D-heap. Data structure for d-heap defined on items numbered [1,n]. Class definition (.h file). Code for operations (.c file). Driver.
  • M-heap. Data structure for "modulo" version of d-heap which compares keys using modular arithmetic. This is intended for use in applications where the key values represent time values that may "wrap-around" and where the range valid key values is always close to the "current time". Class definition (.h file). Code for operations (.c file). Driver.
  • Leftist Heaps Data Structure. Data structure representing a collection of leftist heaps that define a partition on a set of items numbered [1,n]. Class definition (.h file). Code for operations (.c file). Driver.
  • Lazy leftist Heaps Data Structure. Data structure representing a collection of leftist heaps with lazy deletion that define a partition on a set of items numbered [1,n]. Class definition (.h file). Code for operations (.c file). Driver.
  • Fibonacci Heaps Data Structure. Data structure representing a collection of Fibonacci heaps that define a partition on a set of items numbered [1,n]. Class definition (.h file). Code for operations (.c file). Driver.
