00039 // Forest packet format
00040 //
00041 // +----+------------+--------+--------+
00042 // |ver |   length   |  type  |  flags |
00043 // +-----------------------------------+
00044 // |              comtree              |
00045 // +-----------------------------------+
00046 // |              src adr              |
00047 // +-----------------------------------+
00048 // |              dst adr              |
00049 // +-----------------------------------+
00050 // |              hdr chk              |
00051 // +-----------------------------------+
00052 // |                                   |
00053 // |              payload              |
00054 // |                                   |
00055 // +-----------------------------------+
00056 // |            payload chk            |
00057 // +-----------------------------------+
00058 //
00059 // Subscribe/unsubscribe packets have additional fields that
00060 // appear in the payload field.
00061 // Specifically, the first word is an addcnt that specifies
00062 // the number of multicast groups that the sender wants to
00063 // be added to. The next addcnt words are the multicast addresses
00064 // of those groups. The next word following the set of added
00065 // multicast addresses is a drop count. This is followed
00066 // by a list of multicast addresses for groups that the
00067 // sender wants to be removed from.
00068 //
