00001 // Header file for rteTbl class.
00002 //
00003 // Maintains set of tuples of the form (dest, pleng, link1, link2, link3)
00004 // where dest is an IP destination address, pleng specifies the number of
00005 // bits of the address that are relevant here (the prefix length), and the
00006 // the links are possible nexthop links that can be used to reach the destination,
00007 // in order of preference. All are expected to connect to routers
00008 // that are "closer" to the destination than "this router".
00009 // If any of the links is zero, the remaining ones are ignored.
00010 //
