Detailed Description

The CtlPkt class is used to pack and unpack forest control messages. The class has a public field for every field that can be used in in a control packet. To create a control packet, the user constructs a CtlPkt object with a pointer to the payload part of the buffer as the constructor argument. The user then sets selected fields to non-zero values and calls pack, which packs the appropriate fields into the packet's payload and returns the number of 32 bit words that were packed.

To unpack a buffer, the user constructs a CtlPkt object and then calls unpack, specifying the number of 32 bit words in the payload that are to be unpacked. The control packet fields can then be retrieved from the corresponding fields of the CtlPkt object.
